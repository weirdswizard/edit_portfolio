# Editkaro.in Portfolio Page

A standalone portfolio page developed for **Editkaro.in** to showcase video editing work across multiple content categories. The page is designed to provide a clean browsing experience with responsive layouts, smooth interactions, and direct access to featured projects.

## Features

- Responsive portfolio grid for desktop, tablet, and mobile devices
- Category-based filtering for quick project navigation
- Light and dark theme support
- Smooth scroll reveal and hover animations
- Dynamic project rendering using JavaScript
- Automatic YouTube thumbnail loading
- Direct links to YouTube videos
- Modern and minimal user interface
- Accessibility-friendly controls

## Categories

The portfolio includes projects from the following categories:

- Short Form
- Long Form
- Gaming
- Football
- eCommerce
- Documentary
- Color Grading
- Anime
- Ads
